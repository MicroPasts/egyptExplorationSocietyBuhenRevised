Egypt Exploration Society Buhen Season 1958

This project is an application for the crowd-sourcing of data from archival cards that have been generated from the Egypt Exploration Society's Buhen digging season 1958. Each card contains very structured data relating to each object recorded, and these data are what we would like you to transcribe.

Therefore the field list we would like is as follows:

  1. Description
  2. Site code
  3. Object number
  4. Provenance
  5. Distribution/ Museum sent to
  6. Scale
  7. Your comments

Finally, you may have comments on the index card, if you do, please fill in the final box and then submit your contribution. Thank you for taking part.
